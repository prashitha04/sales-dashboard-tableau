#Sales Performance Dashboard (Tableau)

#Overview

This project analyzes 10,000+ sales records using Tableau to uncover meaningful business insights.

#Key Insights

* West region generates the highest revenue
* Technology category contributes the most profit
* Sales peak observed during Q4
* Top 10 customers drive a significant share of revenue

#Tools Used

* Tableau
* Microsoft Excel
